## Title: GetNumBankSlots

**Content:**
Returns the number of purchased bank bag slots.
`numSlots, full = GetNumBankSlots()`

**Returns:**
- `numSlots`
  - *number* - Number of bag slots already purchased.
- `full`
  - *boolean* - True if no further slots are available.

**Reference:**
- `GetBankSlotCost`
