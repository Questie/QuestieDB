## Title: C_Map.GetMapArtHelpTextPosition

**Content:**
Returns the position for the "Click to Zoom In" hint text on flight maps.
`position = C_Map.GetMapArtHelpTextPosition(uiMapID)`

**Parameters:**
- `uiMapID`
  - *number* : UiMapID

**Returns:**
- `position`
  - *Enum.MapCanvasPosition*
    - `Value`
    - `Field`
    - `Description`
    - `0`
      - None
    - `1`
      - BottomLeft
    - `2`
      - BottomRight
    - `3`
      - TopLeft
    - `4`
      - TopRight
