## Title: HasAction

**Content:**
Returns true if an action slot is occupied.
`hasAction = HasAction(actionSlot)`

**Parameters:**
- `actionSlot`
  - *number* - ActionSlot: The tested action slot.

**Returns:**
- `hasAction`
  - *boolean* - Flag
    - `true`, if the slot contains an action
    - `false`, if the slot is empty
