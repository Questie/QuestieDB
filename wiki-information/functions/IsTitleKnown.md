## Title: IsTitleKnown

**Content:**
Returns true if the character can use a player title.
`isKnown = IsTitleKnown(titleId)`

**Parameters:**
- `titleId`
  - *number* - Ranging from 1 to `GetNumTitles`.

**Returns:**
- `isKnown`
  - *boolean* - True if the character can use the specified player title.
