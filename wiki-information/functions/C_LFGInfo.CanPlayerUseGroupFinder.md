## Title: C_LFGInfo.CanPlayerUseGroupFinder

**Content:**
Returns true if the player is allowed to use group finder tools, or false and a reason string if not.
`canUse, failureReason = C_LFGInfo.CanPlayerUseGroupFinder()`

**Returns:**
- `canUse`
  - *boolean*
- `failureReason`
  - *string*
