## Title: IsEquippedItem

**Content:**
Determines if an item is equipped.
`isEquipped = IsEquippedItem(itemID or itemName)`

**Parameters:**
- `itemId`
  - *number* - identifier for each unique item
- `itemname`
  - *string* - localized name of an item

**Returns:**
- `isEquipped`
  - *boolean* - is item equipped
