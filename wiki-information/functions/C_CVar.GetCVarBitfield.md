## Title: C_CVar.GetCVarBitfield

**Content:**
Returns the bitfield of a console variable.
`value = C_CVar.GetCVarBitfield(name, index)`
`= GetCVarBitfield`

**Parameters:**
- `name`
  - *string* : CVar - name of the CVar.
- `index`
  - *number* - Bitfield index.

**Returns:**
- `value`
  - *boolean?* - Value of the bitfield.
