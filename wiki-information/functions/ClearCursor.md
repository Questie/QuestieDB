## Title: ClearCursor

**Content:**
Clears any objects from the cursor.
`ClearCursor()`

**Parameters:**
- None

**Returns:**
- None

**Usage:**
```lua
ClearCursor();
PickUpContainerItem(0,1);
ClearCursor();
PickUpContainerItem(0,1);
```

**Description:**
This function was added in patch 1.12 (The Drums of War).
