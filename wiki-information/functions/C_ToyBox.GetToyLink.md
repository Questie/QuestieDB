## Title: C_ToyBox.GetToyLink

**Content:**
Returns the item link for a toy.
`C_ToyBox.GetToyLink(itemID)`

**Parameters:**
- `itemID`
  - *number* - Numeric ID of the item.

**Returns:**
- `itemLink`
  - *string?* - The toy's localized itemLink, or nil if that itemID is not a toy.
