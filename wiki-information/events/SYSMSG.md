## Event: SYSMSG

**Title:** SYSMSG

**Content:**
Fired when a system message occurs. Gets displayed in the UI error frame (the default red text in the top half of the screen) in the default UI.
`SYSMSG: string, r, g, b`

**Payload:**
- `string`
- `string`
- `r`
  - *number*
- `g`
  - *number*
- `b`
  - *number*
