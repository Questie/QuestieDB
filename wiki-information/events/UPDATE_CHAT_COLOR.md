## Event: UPDATE_CHAT_COLOR

**Title:** UPDATE CHAT COLOR

**Content:**
Fired when the chat colour needs to be updated. Refer to ChangeChatColor for details on the parameters.
`UPDATE_CHAT_COLOR: name, r, g, b`

**Payload:**
- `name`
  - *string* - Chat type
- `r`
  - *number* - red
- `g`
  - *number* - green
- `b`
  - *number* - blue
